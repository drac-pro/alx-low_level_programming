# 0x1C. C - Makefiles

In this project we will learn about make and makefiles, how to write makefiles, how to use them and when to use makefiles.
In our tasks we will make use of [these files](https://github.com/alx-tools/0x1B.c)

## General Objectives

- What are **make**, Makefiles
- When, why and how to use Makefiles
- What are rules and how to set and use them
- What are explicit and implicit rules
- What are the most common / useful rules
- What are variables and how to set and use them
